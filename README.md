# Projet : EduConnect - Plateforme Éducative Collaborative

## Description du Projet

EduConnect est une plateforme éducative collaborative conçue pour connecter étudiants et enseignants, offrant une expérience d'apprentissage interactive et personnalisée. Le projet vise à faciliter l'accès à des cours interactifs, à promouvoir le travail de groupe via des espaces collaboratifs et à permettre un suivi de progression détaillé pour chaque utilisateur.

## Structure du Dépôt

```
projet_educatif/
├── .gitignore
├── README.md
├── organisation_equipe.md
├── planification_trello.md
├── synthese_reflexive.md
└── site_vitrine/
    ├── index.html
    └── assets/
        └── css/
        └── js/
        └── img/
```

*   **`.gitignore`** : Fichier pour ignorer les fichiers et dossiers non pertinents pour le contrôle de version (ex: `node_modules`, `.env`).
*   **`README.md`** : Ce fichier, fournissant une description générale du projet, les instructions de configuration et d'utilisation.
*   **`organisation_equipe.md`** : Document détaillant les rôles, responsabilités et le fonctionnement de l'équipe.
*   **`planification_trello.md`** : Planification des tâches du projet, structurée pour être importée ou consultée comme un tableau Trello.
*   **`synthese_reflexive.md`** : Document de synthèse réflexive sur le travail de groupe et les apprentissages.
*   **`site_vitrine/`** : Contient les fichiers du site web de présentation du projet.
    *   **`index.html`** : La page d'accueil du site vitrine.
    *   **`assets/`** : Dossier pour les ressources statiques (CSS, JS, images).
        *   **`css/`** : Fichiers CSS personnalisés (si nécessaire, en complément de Tailwind CSS).
        *   **`js/`** : Fichiers JavaScript pour les fonctionnalités interactives.
        *   **`img/`** : Images utilisées sur le site vitrine.

## Installation et Lancement (Site Vitrine)

Pour visualiser le site vitrine :

1.  **Cloner le dépôt** :
    ```bash
    git clone [URL_DU_DEPOT]
    cd projet_educatif/site_vitrine
    ```
2.  **Ouvrir `index.html`** :
    Ouvrez simplement le fichier `index.html` dans votre navigateur web préféré.
